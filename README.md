## 功能：
移动互联网开发的课程设计项目，实现了家居人体感应功能。<br>
使用了蜂鸣器，wifi模块和红外感应器，当设置为无人模式时，红外感应器便开始工作，当检测到有物体经过时便开启蜂鸣器报警，并通过wifi模块连接互联网通知用户。<br>

## 使用方法：
输入已设置好的蜂鸣器与红外感应器的ip地址，开启连接即可
